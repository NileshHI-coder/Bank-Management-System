
# Bank Management Application

## 📌 Project Overview
The Bank Management Application is a console-based project developed in C++ using Object-Oriented Programming (OOP) and File Handling concepts.  
This application simulates basic banking operations such as account creation, deposit, withdrawal, balance checking, and storing customer records permanently in files.

---

# 🎯 Objective
To design and implement a secure and functional banking system capable of performing banking operations while maintaining persistent customer records.

---

# 🚀 Features
- Create New Bank Account
- Deposit Money
- Withdraw Money
- Check Account Balance
- Display Customer Details
- Store Customer Data Using File Handling
- Menu Driven Program
- Simple and Easy User Interface

---

# 🛠 Technologies Used
- Language: C++
- Concepts:
  - Classes & Objects
  - Encapsulation
  - File Handling
  - Functions
  - Conditional Statements
  - Loops
